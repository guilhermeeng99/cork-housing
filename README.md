# Cork Housing Catalogue

Catálogo de anúncios de moradia em Cork, Irlanda, coletados do Daft.ie.

- **Etapa 1:** quartos privativos (casa compartilhada) e lugares inteiros (estúdio / 1 quarto).
- **Etapa 2:** imóveis de 2 a 3 quartos, pet-friendly.

Abra `index.html` (autocontido) ou acesse a página publicada via GitHub Pages. Use as abas para alternar entre Etapa 1 e Etapa 2 e os filtros (bairro, preço, pet, etc.).

Dados capturados em 2026-07-06. Fonte: [Daft.ie](https://www.daft.ie). Este é um catálogo pessoal de referência; confirme sempre cada anúncio na fonte antes de contatar.
